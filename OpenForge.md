## What is OpenForge?

OpenForge is an open-source developer collaboration platform designed to explore transparent project workflows and Web3-aware integrations. This prototype focuses on testing wallet connectivity, basic project interactions, and milestone-oriented flows.

Many features are experimental and subject to change.

Before using OpenForge, make sure you have:

* A modern web browser (Chrome, Brave, Firefox)
* A Web3 wallet: **MetaMask**

## Getting Free Sepolia ETH (Test Only)

Since this is a prototype, OpenForge uses **test ETH**, not real money.

### Recommended Faucets:

* Sepolia Faucet by Alchemy
* Sepolia Faucet by Infura

Steps:

1. Copy your wallet address from MetaMask
2. Paste it into a Sepolia faucet
3. Request test ETH
4. Wait a few seconds for the transaction to confirm

You should now see Sepolia ETH in your MetaMask wallet.

## Troubleshooting

**Wallet not connecting?**

* Make sure MetaMask is unlocked
* Confirm you are on the Sepolia network
* Refresh the page and retry

**No ETH showing?**

* Confirm the faucet transaction succeeded
* Check Sepolia is selected in MetaMask
